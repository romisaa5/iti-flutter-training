# Flutter Summer Training - ITI

This repo contains the material for the Flutter Development summer training track at ITI, including session content, apps, and tasks throughout the training period.

## What's inside

Each session has its own folder containing:
- The code/app covered in that session
- The task assigned for that session

```
session_1/
task_1/
session_2/
task_2/
session_3/
task_3/..............
...
```

This repo will be updated regularly as new sessions and tasks are added.
